## ğŸ”‘ Setting Up Your Own API Keys

**Important:** This project requires your own API keys to work. I've removed my personal keys for security reasons, so you'll need to set up your own (don't worry, it's free!).

### Step 1: Create a `.env` file

Create a file named `.env` in the root directory with the following:

```env

VITE_GOOGLE_MAPS_API_KEY=your_google_maps_api_key_here

# Application Configuration

VITE_APP_URL=http://localhost:8080
```

### Step 2: Get Your Google Maps API Key (Free)

1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Create a new project (or use an existing one)
3. Enable these APIs:
   - Maps JavaScript API
   - Places API
   - Directions API
   - Geocoding API
4. Go to **Credentials** â†’ **Create Credentials** â†’ **API Key**
5. Copy your key and paste it in the `.env` file

> ğŸ’¡ **Tip:** Google gives you $200 free credit monthly â€” that's 28,000+ map loads for free!

### Step 3: Set Up Firebase

1. Go to [Firebase Console](https://console.firebase.google.com/)
2. Create a new project called "GreenCommute" (or any name you like)
3. Enable **Authentication** (Email/Password)
4. Enable **Firestore Database**
5. Go to Project Settings â†’ Your Apps â†’ Add Web App
6. Copy the config and update `src/lib/firebase.ts`:

```typescript

  apiKey: "YOUR_FIREBASE_API_KEY",
  authDomain: "your-project.firebaseapp.com",
  projectId: "your-project-id",
  storageBucket: "your-project.appspot.com",
  messagingSenderId: "your-sender-id",
  appId: "your-app-id"

```

### Step 4: Restart and You're Good to Go!

